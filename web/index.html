The Research Team — Replaced
Each agent maps to a real role on a traditional equity research desk.
Agent 01
Data Fusion
Replaces: Junior Analyst
Fires 3 concurrent API calls to SEC EDGAR, Alpha Vantage, and FRED. Normalizes 25+ financial fields. Falls back gracefully if any source fails.
Agent 02
RAG Citation
Replaces: Research Associate
Downloads 10-K filings, parses sections (MD&A, Risk Factors), chunks at 500 tokens, embeds via Gemini, stores in ChromaDB. Retrieves top-k relevant passages.
Agent 03
Quant Analysis
Replaces: Quantitative Analyst
3-stage DCF model (bear/base/bull), RSI(14), MACD(12/26/9), earnings surprise vs consensus. WACC derived from FRED rate + beta.
Agent 04
Risk Scanner
Replaces: Risk & Compliance Officer
Two-stage detection: keyword/regex pattern match across 9 risk categories, then Gemini LLM confirmation of severity. Cites specific filing passages.
Agent 05 · Crown Jewel
Verification
Replaces: Senior Analyst
Cross-references management's narrative claims against quantitative data. Flags divergences with evidence. Assigns confidence scores per section. Forces valuation to MEDIUM.
Agent 06
Report Synthesis
Replaces: Publishing Editor
Assembles 5 structured sections with citations, confidence scores, and overall verdict. Generates follow-up Q&A via grounded Gemini chat.

Quick Start
Running locally takes about 2 minutes.
# Clone and install
git clone https://github.com/your-username/alphalens.git
cd alphalens
pip install -r requirements.txt

# Add your free API keys to .env
GOOGLE_API_KEY=your_key
ALPHA_VANTAGE_API_KEY=your_key
FRED_API_KEY=your_key

# Launch
streamlit run app.py
